# 드림어리 일본어 번역 규칙 사전 (Dreamary Japanese Translation Glossary)

이 문서는 한국어 단어를 일본어로 번역할 때 일관된 세계관(드림 문화)을 유지하기 위해 사용하는 필수 번역 규칙입니다. 앱 내 새로운 텍스트를 번역할 때 반드시 이 규칙을 최우선으로 적용하세요.

## 핵심 용어 매핑
| 한국어 | 일본어 번역 | 비고 / 설명 |
| --- | --- | --- |
| 드림캐 (추천캐) | 夢キャ | 推しキャラ(오시캐라) 대신 사용 |
| 드림주 (유저) | 夢主 | |
| 서사 | 関係性 / ストーリー / エピソード / 設定 | 문맥에 따라 자연스러운 단어 선택. 예: 서사를 입력해주세요 -> 関係性を入力してください / 몽주와의 서사 -> 夢主とのストーリー |
| 캐입 (캐릭터 이입) | なりきり / キャラクターらしく話すため | "캐입을 위해~" 문맥에서는 "キャラクターらしく話すため" 사용 권장 |
| 캐붕 (캐릭터 붕괴) | キャラ崩壊してしまう場合は | "キャラ崩壊が起きたら" 등 직접적인 표현보다 약간 완화된 표현 사용 권장 |
| 추가 설정 | 補足設定 | 追加設定 대신 사용 |
| 내 페어 | 私のペア | マイペア 대신 사용 |
| 작품의 세계관을 작성해주세요 | 作品の世界観を書いてください | "ジャンル名ではなくジャンルの世界観" 대신 사용 |
